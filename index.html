⚙️ Settings

Supabase Connection

Same project as Care Coordinator's Hub and Staffing Hub — same staff logins, shared app_data table.

Hub Links

Where the four quick-link buttons point. Update here if any hub's domain ever changes. Lead Caregiver's Hub and Nurse's Hub aren't built yet — their links are placeholders until those hubs are live.

Video Meeting Room

The team's video room is created automatically the first time anyone clicks "Start / Join Video Meeting" — no setup needed. Only regenerate it if the room link has been shared somewhere it shouldn't have been; anyone with the old link loses access once you do.

Team Access Automation

The Zapier webhook that actually sends invite emails and grants hub access (see Hub_Access_Zapier_Setup_Guide.md). Until this is set, Manage Team Access still tracks who should have access, it just can't send the invite itself.

Admin Passcode

Keep this separate from staff login passwords.
